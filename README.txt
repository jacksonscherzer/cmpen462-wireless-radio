Instructions for code execution:

1. Ensure you have Python installed, along with required libraries math, numpy, and tkinter
- math is a default Python library
- numpy can be installed with pip install numpy (Windows) or brew install numpy (Mac)
- tkinter is a default Python library, but often times can be missing on Mac versions. It can be installed using brew install tcl-tk and brew install python-tk

2. Run the Python file either by clicking "run" in an IDE (VSCode) or the command "python3 receiever.py" in the project directory.

3. You will first be prompted to select the 'input.txt' file to use as input. You can use the popup window to browse your computer and select the file.

4. You will then be prompted to select the 'preamble.txt' file to use as the preamble. You can use the popup window to browse your computer and select the file.

5. In your command line window, you will be prompted to enter the carrier frequency, sampling rate, and cutoff frequency in that order. These entries must be numbers only.

6. The decoded message will be printed in the terminal.
